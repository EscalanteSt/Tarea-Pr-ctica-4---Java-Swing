# Calculadora Extendida y Conversor (Java Swing)

Aplicación de escritorio con GUI en Java aplicando POO.

## Funciones
- **Aritmética:** Suma, resta, multiplicación, división (con validación de cero).
- **Avanzadas:** Raíz cuadrada y exponentes.
- **Conversor:** Celsius ↔ Fahrenheit.

## Estructura
- `Calculadora.java`: Lógica básica.
- `ConversorTemperatura.java`: Fórmulas de temperatura.
- `Raizcuadrada_Exponente.java`: Lógica avanzada.
- `Main.java`: Interfaz GUI con JFrame.

## Ejecución
Compilar y ejecutar `Main.java` en cualquier IDE compatible con Java (ej. NetBeans).
